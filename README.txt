V0.1版本程序    

程序使用注意事项（可能遇到的问题）：

该程序基于IntelliJ IDEA 专业版开发。

关于图像存储位置，写在 UploadController 中，可以根据自己需要进行更改；

由于程序并未部署好，如需查看现在效果请直接通过运行程序来自行查看，推荐使用IDE：IntelliJ IDEA；

人员数据暂时存在MySQL数据库中，如感兴趣想要测试，请确保电脑中安装MySQL 并根据程序建立库，
在MySQL建表方面，数据库名为：zc 表名为：person 。如需测试，可以自行建立新的数据库与表，建表语句写在sql文件夹下。
请保证命名方式与mapper中写的对应。

初版程序，可能会存在一些开发测试中不曾遇到的BUG，后续工作会对具体代码进行优化，一切都是为了处理速度更加迅速。
